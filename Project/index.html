Hi, I'm Ken.

Styleme was a cool website. Think, if you can try a clothes online without any effort, why not just stay at home and be yourselves. Styleme covers different clothing categories, scanned clothes and create 3D avatars for visualizations. You can choose your own avatar and match body size you are, and Bang, everything done. The clients include Bloomingdales and Heart of Sun.

I was a software engineer in Styleme, being resiposible for back-end server using Java and some framworks. Because the high quality of images transporting between front-end and back-end, I used zero copy strategy to reduce file-loading time by 65%. It reduce CPU coping time and the number of context switches between user space and kernal space in file system. In addtion, by implementing HTTP caching machenism which sent less requests to our server, I increased image loading speed 50%.
